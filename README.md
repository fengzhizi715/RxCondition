# 下载安装
Gradle:

```groovy
compile 'tony-common:rxcondition:1.0.0'
```

Maven:

```groovy
<dependency>
  <groupId>tony-common</groupId>
  <artifactId>rxcondition</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
